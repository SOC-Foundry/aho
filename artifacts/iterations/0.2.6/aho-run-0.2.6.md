## Context

0.2.6 is a live-fire hardening iteration. After 0.2.5 delivered install.fish, Kyle ran it on NZXTcos and hit three issues:

1. **CachyOS ollama package corruption** — pacman mirror served invalid PGP signatures. Ollama was already installed via upstream script (`/usr/local/bin/ollama`), so removing it from `pacman-packages.txt` was the correct fix.
2. **bin/aho-secrets-init checking wrong store** — the 0.2.5 scaffold checked for `.age` files that don't exist. Telegram secrets live in the fernet secrets store (`~/.config/aho/secrets.fish.fernet`). Rewritten to detect the real store.
3. **Doctor showed no telegram bot identity** — Kyle wanted `aho doctor` to return the actual bot name for verification. Added `getMe` API call cached to state file by the daemon on startup.

All three were discovered during the install.fish idempotency pass. install.fish now completes all 9 steps clean on NZXTcos.
